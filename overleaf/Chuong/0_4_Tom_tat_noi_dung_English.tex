    \Large{\textbf{ABSTRACT}}\\
\end{center}
\vspace{1cm}
Vietnamese is a tonal language with rich diacritics and compound words, which makes Vietnamese word games different from common English crossword games. Existing crossword implementations often focus on English or only display Vietnamese text without carefully handling normalization, diacritics, answer checking, topic organization, and user progress. This thesis presents the design and implementation of a Vietnamese word-crossing game built with the Godot engine. The system organizes vocabulary by topic and stage, normalizes Vietnamese words for board placement, removes spaces when generating the puzzle grid, and still accepts complete Vietnamese answers from the player. The game includes an automatic board generator, a themed level provider, clue management, a responsive game board, local player profiles, score calculation, star-based hints, mission tracking, and saved progress for each account. The final result is a playable prototype that supports multiple Vietnamese topics, generates compact crossword-style boards, adapts the board layout to different screen sizes, and provides a clear play flow from account selection to topic and stage selection. The main contribution of the thesis is a practical workflow for handling Vietnamese vocabulary in a crossword game and integrating that workflow into a complete Godot-based application.
